# Regression & Time Series Analysis

This repository contains an end-to-end data analysis project focused on business sales prediction. It utilizes Python to perform statistical modeling, regression analysis, and ARIMA forecasting to understand trends and predict future performance.

## 📊 Project Overview
The goal of this project is to analyze historical sales data to identify profit drivers and forecast future sales trends.

- **Data Analysis:** Processed raw sales data to uncover key metrics.
- **Regression Modeling:** Identified factors influencing sales and profit.
- **Time Series Forecasting:** Used ARIMA models to predict future sales performance.
